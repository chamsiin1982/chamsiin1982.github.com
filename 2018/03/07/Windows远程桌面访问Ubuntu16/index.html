【译】Windows远程桌面访问Ubuntu 16.04

以前连接Ubuntu远程桌面都是使用的xrdp，要在Ubuntu上安装一堆软件，实际Ubuntu 16.04 提供了远程连接 Desktop Shareing 结合 VNC就能解决

原贴：http://ubuntuhandbook.org/index.php/2016/07/remote-access-ubuntu-16-04/

主要介绍通过启用/配置 VNC服务，使得Ubuntu 16.04 支持远程桌面访问，具体步骤

1.) Ubuntu桌面版提供了 VNC 服务，可通过 Desktop Sharing来配置

2.）配置项包括

共享

  • 允许远程访问当前桌面
  • 运行远程访问者控制桌面（可选）

安全

  • 远程访问需要确认
  • 需要独立密码远程访问
  • 通过公网访问，如果当前路由器开启了UPnP

提醒显示远程连接图标

  • 一直显示
  • 有远程连接时
  • 从不显示

3.）关闭加密

直接使用TigerVNC Viewer连接服务时，会弹出不支持的安全类型，需关闭安全选项

可通过 DCONF Editor配置“远程访问“ 需加密选项 关闭

路径org -> gnome -> desktop -> remote-access

客户端再次连接，可以访问
